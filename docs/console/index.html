Deployment rule

GitHub Pages hosts only immutable static assets. Any backend must be independently deployed with authentication, explicit CORS origins, least-privilege credentials, audit logging, rate limits, and deny-by-default action policy.

Recommended adapter sequence

  1. Public repository and documentation reads.
  2. Authenticated evidence and workflow reads.
  3. Reviewable draft actions with no automatic execution.
  4. Explicitly approved writes through a policy gateway.
